# Simulador de Cotización de Seguros

Cotizador interactivo para seguros de hogar desarrollado con React.

## Descripción del proyecto

Este proyecto es un simulador profesional desarrollado para la diplomatura de Frontend. Integra una interfaz moderna y funcional con las siguientes tecnologías:

- **React JS + Vite**: Estructura de componentes eficiente.
- **JavaScript Moderno**: Uso intensivo de ES6+ y Sugar Syntax.
- **Fetch API**: Carga de datos dinámicos para las cotizaciones.
- **Local Storage**: Persistencia de datos para el historial de cotizaciones.
- **SweetAlert2**: Integración de ventanas emergentes interactivas para el borrado y confirmación.
- **CSS Avanzado**: Diseño responsive, footer alineado y botones estilizados.

## Características principales

- Selección de tipo de propiedad y ubicación.
- Cálculo de póliza en tiempo real.
- Historial de cotizaciones con opción de borrado individual.
- Interfaz amigable y profesional.
